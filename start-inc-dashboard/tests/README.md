# Start Inc. Dashboard - Test Suite

This directory contains automated tests and scripts for verifying the integrity of the dashboard.

## Structure
- `api_check.sh`: Shell script to verify backend API health.
- `e2e/`: End-to-end tests using Playwright (Planned).
- `integration/`: Integration tests for key flows (Planned).

## Usage
Run the API check:
```bash
./api_check.sh
```
